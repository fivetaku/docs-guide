## URL Patterns to Try

When a library is not in either list above, try these patterns in order:

1. `{official-site}/llms.txt`
2. `{official-site}/docs/llms.txt`
3. `{official-site}/llms-full.txt`
4. `{official-site}/docs/llms-full.txt`

## How to Verify

Fetch the URL with WebFetch. A valid llms.txt:
- Returns HTTP 200
- Contains structured text with links to documentation pages
- May include section headers, page titles, and URLs
- Note: some sites redirect (e.g., docs.anthropic.com → platform.claude.com). Follow redirects.
